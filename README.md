# Smart Weather Station AI Lab

Smart Weather Station AI Lab is a long-term AI engineering portfolio project that connects embedded systems, environmental sensing, data analysis, machine learning, Edge AI and technical documentation.

The goal is to build a low-cost weather station using recycled materials and environmental sensors, then use the collected data to learn and apply Python, statistics, Machine Learning, Deep Learning, Computer Vision, Edge AI and LLM-based reporting.

## Project status

Current version: **v0.0 — Project setup and Python environment**

At this stage, the project includes:

- Python virtual environment setup
- Initial project structure
- Basic environment test script
- NumPy, pandas and matplotlib validation
- VS Code configuration
- Debugger setup

## Long-term goal

The final goal is to create a physical AI-assisted environmental monitoring system that can:

- collect real sensor data,
- process and clean data with Python,
- visualize environmental trends,
- train Machine Learning models,
- detect anomalies,
- run lightweight inference on edge hardware,
- generate technical reports automatically.

## Planned hardware

Initial hardware candidates:

- ESP32 or Raspberry Pi Pico W
- Temperature and humidity sensor
- Pressure sensor
- Light sensor
- Recycled enclosure
- USB power supply
